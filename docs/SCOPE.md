# Scope

## Page Structure

Single-page scrolling proposal. All content is presented in sequential sections.

## Sections

1. **Hero** -- introduction and value proposition
2. **Health Scorecard** -- overall digital health summary
3. **Executive Summary** -- key findings at a glance
4. **Meta Ads Audit** -- Facebook/Instagram advertising analysis
5. **Google Ads Audit** -- Search and display advertising analysis
6. **Web Performance** -- Core Web Vitals and page speed findings
7. **Wasted Spend Calculator** -- interactive estimate of inefficient spend
8. **Action Plan** -- prioritised recommendations
9. **Next Steps / CTA** -- call-to-action for strategy session booking

## Excluded

- Internal dashboards or reporting tools
- Live data connections or API integrations
- Content management system (CMS)
- Authentication or user accounts
- E-commerce functionality

## Launch Milestones

1. **Build** -- develop all sections and finalise audit content
2. **Review** -- internal QA and content review
3. **Deploy** -- push to Vercel production
4. **Share** -- send proposal link to client
