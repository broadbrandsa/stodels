# Project Overview

## What This Is

A single-page proposal website presenting a digital marketing audit for **Stodels Nursery (Pty) Ltd**, a South African retail garden centre with e-commerce capabilities. The site is built and owned by **Broadbrand**.

## Target Audience

Stodels marketing team and key decision-makers. The proposal is designed to communicate audit findings clearly and drive a strategy session booking.

## Tech Stack

| Layer        | Technology                        |
| ------------ | --------------------------------- |
| Framework    | Next.js (App Router)              |
| Language     | TypeScript                        |
| Styling      | Tailwind CSS v4                   |
| Components   | shadcn UI                         |
| Deployment   | Vercel                            |
| Package Mgr  | pnpm                              |

## Deployment

Hosted on Vercel. Pushes to the `main` branch trigger automatic deployments.
